# Name Game Dashboard project  

This projects aims to create a R Shiny dashboard that will allow to play remotely a popular party game named the Name Game.  
The dashboard uses Google Drive docs to set up players and their teams, as well as deliver names to guess to current players.

### Requirements:  
To use this project, you'll need:
* `R` installed with packages such as `shiny` - recommended to use package `renv` and `renv.lock` file provided to install all packages used in the project  
* `RStudio` or `Shiny-server` to host the dashboard  
* Google Drive account
